Movie Recommendation System

Python ile geliştirilmiş içerik tabanlı (content-based) film öneri sistemi.

Kullanılan Teknolojiler
- Python
- Pandas
- Scikit-learn

Nasıl Çalışır?
Film verileri TF-IDF ile vektörleştirilir ve cosine similarity kullanılarak benzer filmler bulunur.

Nasıl Çalıştırılır?
1. Gerekli kütüphaneleri yükleyin:
pip install -r requirements.txt

2. Programı çalıştırın:
python app.py

Amaç
Bu proje, öneri sistemlerinin temel mantığını öğrenmek amacıyla geliştirilmiştir.
